# 🚨 ZeroTouch Rescue  
### *Automated Crisis Response System | Google Solution Challenge 2026*

ZeroTouch Rescue is a next-generation safety application designed to minimize the delay between an emergency and rescue response. Unlike conventional SOS apps that depend on manual input, ZeroTouch operates intelligently in the background to detect distress situations and trigger immediate assistance.

---

## 🎯 Problem Statement

In real-world emergencies such as accidents, medical distress, or personal threats, victims often cannot access their phones. This critical delay can cost lives.

### 💡 Our Solution
ZeroTouch Rescue transforms a smartphone into an intelligent safety companion that automatically detects emergencies using sensors and voice recognition—ensuring help is triggered even when the user cannot act.



## ✨ Key Features

### 🆘 Smart Emergency Detection
- Voice-triggered alerts using keywords like “Help”, “Kapadandi”, “Amma”
- Real-time fall detection using accelerometer & gyroscope

### 📍 Instant Emergency Response
- Automatic SMS alerts to emergency contacts
- Live GPS location sharing
- Optional loud alarm activation

### 🧠 Background Intelligence
- Continuous sensor monitoring in background mode
- Optimized for low battery usage

### 🎨 Clean & Accessible UI
- Simple dashboard for quick access
- Calm and user-friendly design for emergency situations

## ⚙️ Setup & Installation

To run this project securely, follow these steps:

### 1. Configure Environment Variables 🔐
We use the `flutter_dotenv` package to keep API keys secure.
1. Create a file named `.env` in the **root directory** of the project.
2. Add your Google Maps API key to the file:
   ```text
   GOOGLE_MAPS_API_KEY=your_actual_key_here
